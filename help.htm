Commands and Instructions in Befunge
====================================

0-9 Push this number on the stack
 + Addition: Pop a and b, then push a+b
 - Subtraction: Pop a and b, then push b-a
 * Multiplication: Pop a and b, then push a*b
 / Integer division: Pop a and b, then push '\'b/a, rounded down. If a'' is zero, ask the user what result they want.
 % Modulo: Pop a and b, then push the remainder of the integer division of b/a. If a is zero, ask the user what result they want.
 ! Logical NOT: Pop a value; if the value is zero, push 1, otherwise zero.
 ` Greater than: Pop a and b, then push 1 if b>a, otherwise zero.
 > Move right
 < Move left
 ^ Move up
 v Move down
 ? Move in a random direction
 _ Pop a value; move right if value=0, left otherwise
 | Pop a value; move down if value=0, up otherwise
 " Start string mode: push each character's ASCII value all the way up to the next "
 : Duplicate value on top of the stack
 \ Swap two values on top of the stack
 $ Pop value from the stack
 . Pop value and output as an integer
 , Pop value and output as ASCII character
 # Trampoline: Skip next cell
 g Pop y and x, then pushes ASCII value of the character at that position in the program
 p Pop y, x and v, then change the character at the position x/y in the program to the character with ASCII value v
 & Ask user for a number and push it
 ~ Ask user for a character and push ASCII value
 @ End program

Anweisungen in Befunge 93
=========================

0-9 Legt diese Nummer auf den Stapel

+ Addition - Entfernt a und b und legt stattdessen b+a auf den Stapel
- Subtraktion - Entfernt a und b und legt stattdessen b-a auf den Stapel
* Multiplikation - Entfernt a und b und legt stattdessen b*a auf den Stapel
/ Ganzzahldivision - Entfernt a und b und legt stattdessen das ganzzahlige, abgerundete Ergebnis von b\a auf den Stapel
% Modulo (Rest) - Entfernt a und b und legt stattdessen den ganzzahligen Rest der Rechnung b\a auf den Stapel

! Logische Negation (NOT) - Entfernt den obersten Wet vom Stapel; ist dieser gleich 0, wird 1 auf den Stapel gelegt, sonst 0
` Größer als - Entfernt a und b und legt 1 auf den Stapel, falls b>a ist, andernfalls 0.

> Bewegungsrichtung: Osten
< Bewegungsrichtung: Westen
^ Bewegungsrichtung: Norden
v Bewegungsrichtung: Süden

? zufällige Bewegungsrichtung (Osten, Westen, Norden oder Süden)

_ Entfernt den obersten Wert vom Stapel; ist dieser gleich 0, Ist die neue Bewegungsrichtung Osten, sonst Westen
| Entfernt den obersten Wert vom Stapel; ist dieser gleich 0, Ist die neue Bewegungsrichtung Süden, sonst Norden

" Startet den Zeichenkettenmodus: der ASCII-Wert jedes folgenden Zeichens wird auf den Stapel gelegt, bis zum nächsten "

# Trampolin: Überspringt die nächste Befehlszelle

: Dupliziert Den obersten Wert des Stapels
\ Vertauscht die beiden obersten Werte auf dem Stapel
$ Entfernt den obersten Wert vom Stapel
. Entfernt den obersten Wert vom Stapel und gibt ihn als Zahl aus
, Entfernt den obersten Wert vom Stapel und gibt ihn als ASCII-Zeichen aus
g Entfernt y und x vom Stapel und schreibt den ASCII-Wert des Zeichens an der Position x,y im Program auf den Stapel
p Entfernt y, x und v vom Stapel und schreibt das Zeichens mit dem ASCII-Wert v an der Position x,y ins Program
& Erzwingt die Benutzereingabe einer Zahl, die dann auf den Stapel geschrieben wird
~ Erzwingt die Benutzereingabe eines ASCII-Zeichens, welches dann auf den Stapel geschrieben wird
